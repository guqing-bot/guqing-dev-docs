## 图片

### 使用 Markdown

[markdown 语法](https://www.markdownguide.org/basic-syntax/#images)允许您使用以下代码添加图片

```md
![标题](/path/image.jpg)
```

请注意，图片文件大小必须小于 5MB。否则，我们建议在 [Cloudinary](https://cloudinary.com/) 或 [S3](https://aws.amazon.com/s3/) 等服务上托管。然后您可以使用该 URL 并嵌入。

### 使用嵌入

为了获得图片的更多自定义功能，您也可以使用[嵌入](/writing-content/embed)来添加图片

```html
<img height="200" src="/path/image.jpg" />
```

<Tip>

Poetora 支持 [Markdown 中的 HTML 标签](https://www.markdownguide.org/basic-syntax/#html)。如果您更喜欢 HTML 标签而不是 Markdown 语法，这很有帮助，并且让您可以创建具有无限灵活性的文档。

</Tip>

### iframe

在文档中加载另一个 HTML 页面。最常用于嵌入视频。

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
