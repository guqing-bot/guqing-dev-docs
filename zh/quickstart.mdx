## 三步入门

让您的文档站点在本地运行并进行第一次自定义。

### 步骤 1：设置本地环境

<Accordions type="single">  
  <Accordion title="本地克隆您的文档" icon="copy">
    在入门过程中，如果您还没有 GitHub 存储库，我们会为您创建一个包含文档内容的存储库。您可以在[控制面板](https://dashboard.poetora.com)中找到此存储库的链接。

    要在本地克隆存储库以便您可以对文档进行更改和预览，请遵循 GitHub 文档中的[克隆存储库](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)指南。
  </Accordion>
  <Accordion title="启动预览服务器" icon="rectangle-terminal">
    1. 安装 Poetora CLI：`npm i -g mint`

    2. 导航到您的文档目录并运行：`mint dev`

    3. 打开 `http://localhost:3000` 查看您的实时文档！

    <Tip>
    当您编辑文件时，预览会自动更新。
    </Tip>
  </Accordion>
</Accordions>

### 步骤 2：部署您的更改

<Accordions type="single">  
  <Accordion title="安装我们的 GitHub 应用" icon="github">
    从您的[控制面板](https://dashboard.poetora.com/settings/organization/github-app)安装 Poetora GitHub 应用。

    我们的 GitHub 应用会自动将您的更改部署到文档站点，因此您无需自己管理部署。
  </Accordion>
  <Accordion title="更新您的站点名称和颜色" icon="palette">
    作为第一次更改，让我们更新您文档站点的名称和颜色。

    1. 在编辑器中打开 `docs.json`。

    2. 将 `"name"` 字段更改为您的项目名称。

    3. 更新 `"colors"` 以匹配您的品牌。

    4. 保存并在 `http://localhost:3000` 立即查看您的更改。

    <Tip>
    尝试更改主要颜色以立即看到不同！
    </Tip>
  </Accordion>
</Accordions>

### 步骤 3：上线

<Accordion title="发布您的文档" icon="rocket">
  1. 提交并推送您的更改。 2. 您的文档将在几分钟内更新并上线！
</Accordion>
